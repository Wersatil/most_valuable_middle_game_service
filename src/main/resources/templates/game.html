Игроки

1. Любое определение должно начинаться с существительного. Например: "Что такое @Component" - "Это - аннотация Spring, которая..."

2. За полный правильный ответ игрок получает 3 балла.

3. За 50% ответа игрок получает 1 балл.

4. Если игрок дает менее 50% ответа, он не получает баллов, и ответить на вопрос получает право один другой игрок (приоритет получают те, у кого меньше баллов).

5. За полный ответ вне своей очереди игрок получает 2 балла, за неполный ответ не получает ничего.
